# Face Recognition Based Attendance System

This project is a face recognition-based attendance system. It uses computer vision library OpenCV to recognize faces and track attendance.

## Features

- Face detection and recognition
- Attendance tracking
- User-friendly interface
- Real-time updates

## Installation

1. Clone the repository:

    ```bash
    https://github.com/hdksrma/FaceRecognitionAttendance
    ```

2. Install the required dependencies:

    ```bash
    pip install flask pandas opencv-python numpy sklearn
    ```

3. Run the application:

    ```bash
    python app.py
    ```

## Usage

1. Launch the application.
2. Capture images of individuals to create a face database.
3. Start the attendance tracking process.
4. View attendance reports.
